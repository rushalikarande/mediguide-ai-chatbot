# MediGuide AI Chatbot

MediGuide is an AI-powered chatbot developed using Botpress to assist users with medication guidance and meal support recommendations through interactive conversations.

## Features

- Voice assistant integration
- Multi-language support
- Better AI response handling
- AI-based chatbot interaction
- Medication support assistance
- Meal guidance recommendations
- Automated conversation flow
- User-friendly interface
- Interactive responses

## Technologies Used

- Botpress
